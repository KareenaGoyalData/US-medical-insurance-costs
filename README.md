# U.S. Medical Insurance Costs Analysis

## Overview
This project explores what factors most strongly predict medical insurance charges 
in the U.S. using exploratory data analysis and linear regression modeling.

## Key Findings
- Smoking status was the strongest single predictor of insurance charges
- Smokers with a BMI ≥ 30 were charged $19,454.94 more on average — the highest 
  cost group in the dataset
- Adding a smoker × BMI interaction term improved model accuracy from 81% to 91%
- Age, number of children, and BMI had smaller but meaningful effects on charges
- Region and sex had minimal independent effects on charges
